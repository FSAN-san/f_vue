# About
This is a `Vue 3 + TypeScript + Vite` project.

The plugin is as follows:
1. `EsLint`
2. `Vue Router`
3. `Vuex`
4. `PostCSS`
5. `SCSS`

# Use
此项目使用包管理为`yarn`，开发工具为`WebStorm`，请使用`21年3月`以上版本

## 为WebStorm配置ESLint环境
`Settings`中搜索`eslint`选择第二个`automatic`即可

## 下载依赖及启动
```bash
$ yarn
$ yarn dev
```
